# Cinema Schedule


###About
This is project show you cinema schedule. 
You can choose move and then watch detail information about this move

###Frameworks
<ul>
    <li>Spring Security</li>
    <li>Freemarker</li>
    <li>Swagger 2.9.2</li>
    <li>JPA</li>
</ul>

###Database
